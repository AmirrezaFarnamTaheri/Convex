Key Concepts

Concept 1: The Optimization Problem

Detailed explanation with inline LaTeX: An optimization problem has the form $\min_{x \in \mathcal{D}} f(x)$ subject to $g_i(x) \le 0$ and $h_j(x) = 0$.

Visual intuition: We are trying to find the lowest point in a landscape.

Concept 2: Convexity

A set is convex if the line segment between any two points in the set is also in the set.
