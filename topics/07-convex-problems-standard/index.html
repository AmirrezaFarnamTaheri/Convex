07. Convex Optimization Problems: Standard Forms

Date: 2025-11-11 Duration: 90 min Tags: standard-forms, classification, LP, QP

Overview: This lecture covers the standard form of convex optimization problems and the fundamental problem classes: Linear Programs (LP) and Quadratic Programs (QP). We develop techniques for recognizing and formulating convex problems, and introduce problem reformulation methods.

Prerequisites: Lecture 06: Convex Functions Advanced (convex functions, first- and second-order conditions).

Forward Connections: Conic programming (SOCP, SDP) is covered in Lecture 08. Duality theory (Lecture 09) provides optimality conditions for these problem classes.

Learning Objectives

After this lecture, you will be able to:

  • Define and recognize the standard form of a convex optimization problem
  • Classify problems into LP and QP categories with precise criteria
  • Formulate real-world problems using standard forms
  • Apply problem reformulation techniques to convert non-standard forms
  • Make informed decisions about solver selection based on problem structure

1. The Standard Form of Convex Optimization

1.1 Definition

A convex optimization problem in standard form is:

$ \begin{aligned} \text{minimize} \quad & f_0(x) \\ \text{subject to} \quad & f_i(x) \le 0, \quad i = 1, \dots, m \\ & h_j(x) = 0, \quad j = 1, \dots, p \end{aligned} $

where:

  • The variable is $x \in \mathbb{R}^n$
  • $f_0: \mathbb{R}^n \to \mathbb{R}$ is the objective function (convex)
  • $f_i: \mathbb{R}^n \to \mathbb{R}$, $i = 1, \dots, m$ are inequality constraint functions (all convex)
  • $h_j: \mathbb{R}^n \to \mathbb{R}$, $j = 1, \dots, p$ are equality constraint functions (all affine, i.e., $h_j(x) = a_j^\top x - b_j$)

1.2 The Feasible Set

The feasible set (or constraint set) is:

$ \mathcal{F} = \{x \in \mathbb{R}^n \mid f_i(x) \le 0, \; i = 1, \dots, m; \; h_j(x) = 0, \; j = 1, \dots, p\} $

Theorem: Convexity of Feasible Set

Statement: If $f_i$ are convex and $h_j$ are affine, then $\mathcal{F}$ is a convex set.

Proof: The feasible set $\mathcal{F}$ is the intersection of sublevel sets $\{x \mid f_i(x) \le 0\}$ (convex by Lecture 05) and affine sets $\{x \mid h_j(x) = 0\}$ (convex by definition). Since the intersection of convex sets is convex (see Lecture 03), $\mathcal{F}$ is convex.

1.3 Optimal Value and Optimal Points

  • The optimal value is $p^* = \inf\{f_0(x) \mid x \in \mathcal{F}\}$
  • $x^*$ is optimal (or a minimizer) if $x^* \in \mathcal{F}$ and $f_0(x^*) = p^*$
  • $x$ is $\epsilon$-suboptimal if $x \in \mathcal{F}$ and $f_0(x) \le p^* + \epsilon$
  • $x$ is locally optimal if there exists $R > 0$ such that $f_0(x) = \inf\{f_0(z) \mid z \in \mathcal{F}, \|z - x\|_2 \le R\}$

Fundamental Property: Local = Global

Statement: For convex optimization problems, any locally optimal point is globally optimal.

Proof: This is a consequence of the convexity of $f_0$ and $\mathcal{F}$, as established in Lecture 02. If $x$ is locally optimal but not globally optimal, we can construct a line segment from $x$ to a better point that contradicts local optimality.

2. Linear Programs (LP)

2.1 Definition

A Linear Program (LP) has the form:

$ \begin{aligned} \text{minimize} \quad & c^\top x \\ \text{subject to} \quad & Gx \preceq h \\ & Ax = b \end{aligned} $

where:

  • $x \in \mathbb{R}^n$ is the variable
  • $c \in \mathbb{R}^n$ is the cost vector
  • $G \in \mathbb{R}^{m \times n}$, $h \in \mathbb{R}^m$ define the inequality constraints
  • $A \in \mathbb{R}^{p \times n}$, $b \in \mathbb{R}^p$ define the equality constraints
  • $\preceq$ denotes componentwise inequality

2.2 Key Properties

  • Feasible set: A polyhedron (intersection of halfspaces and hyperplanes)
  • Optimal solution: If it exists and the optimal value is finite, occurs at a vertex of the polyhedron
  • Complexity: Polynomial-time solvable (simplex method, interior-point methods)
  • Duality: Strong duality always holds, a property we will prove in Lecture 09 using the Separating Hyperplane Theorem.

2.3 Standard LP Examples

Example 2.1: Diet Problem

The goal is to choose quantities of foods to meet nutrient requirements at minimum cost.

  • Variables: $x_j$ is the amount of food $j$ to consume (for $j=1,\dots,n$).
  • Data:
    • $c_j$: Cost per unit of food $j$.
    • $A_{ij}$: Amount of nutrient $i$ in one unit of food $j$.
    • $b_i$: Minimum required amount of nutrient $i$ (for $i=1,\dots,m$).

The problem is formulated as:

$ \begin{aligned} \text{minimize} \quad & \sum_{j=1}^n c_j x_j \\ \text{subject to} \quad & \sum_{j=1}^n A_{ij} x_j \ge b_i, \quad i = 1, \dots, m \\ & x_j \ge 0, \quad j = 1, \dots, n \end{aligned} $

This is a standard LP. The non-negativity constraints $x_j \ge 0$ are essential (we cannot consume negative food).

Example 2.2: Chebyshev Approximation

Given a system of linear equations $Ax=b$ that has no solution (overdetermined), we might want to minimize the maximum absolute error (Chebyshev norm or $\ell_\infty$-norm).

Problem: $\min_x \|Ax - b\|_\infty = \min_x \max_i |a_i^\top x - b_i|$.

This objective is convex (pointwise max of convex functions) but not linear. We reformulate it as an LP by introducing an epigraph variable $t$:

$ \begin{aligned} \text{minimize} \quad & t \\ \text{subject to} \quad & -t \mathbf{1} \preceq Ax - b \preceq t \mathbf{1} \end{aligned} $

The constraints expand to $-t \le a_i^\top x - b_i \le t$ for all $i$. This is an LP in variables $(x, t)$.

2.4 Piecewise-Linear Minimization

Minimizing $f(x) = \max_{i=1,\dots,m} (a_i^T x + b_i)$ (a piecewise-linear convex function) can be reformulated as:

$ \begin{aligned} \text{minimize} \quad & t \\ \text{subject to} \quad & a_i^T x + b_i \le t, \quad i = 1, \dots, m \end{aligned} $

This transformation is fundamental: many convex problems can be reduced to LPs via epigraph reformulation.

3. Quadratic Programs (QP)

3.1 Definition

A Quadratic Program (QP) has the form:

$ \begin{aligned} \text{minimize} \quad & \frac{1}{2} x^\top P x + q^\top x + r \\ \text{subject to} \quad & Gx \preceq h \\ & Ax = b \end{aligned} $

where $P \in \mathbb{S}_+^n$ (positive semidefinite), $q \in \mathbb{R}^n$, $r \in \mathbb{R}$.

Convexity of QP

Objective convexity: Since $P \succeq 0$, the quadratic form $f_0(x) = \frac{1}{2} x^\top P x + q^\top x + r$ is convex. This follows from the second-order condition: $\nabla^2 f_0(x) = P \succeq 0$.
Feasible set convexity: The constraints define a polyhedron, which is convex.
Conclusion: A QP is a convex optimization problem.

3.2 Key Properties

  • Optimality: If $P \succ 0$ (positive definite), the problem is strictly convex and has a unique minimizer (if feasible)
  • KKT conditions: Necessary and sufficient for optimality (Lecture 09). For QPs, these are linear equations, making them very fast to solve.
  • Solvers: Interior-point methods, active-set methods

3.3 Standard QP Examples

Example 3.1: Least-Squares with Linear Inequality Constraints

Minimize $\|Ax - b\|_2^2$ subject to $Cx \preceq d$:

$ \begin{aligned} \text{minimize} \quad & x^\top A^\top A x - 2 b^\top A x + b^\top b \\ \text{subject to} \quad & Cx \preceq d \end{aligned} $

This is a QP with $P = 2A^\top A \succeq 0$ and $q = -2A^\top b$. Common constraints include non-negativity ($x \ge 0$) or box constraints ($l \le x \le u$).

Example 3.2: Markowitz Portfolio Optimization

We want to find an asset allocation $x \in \mathbb{R}^n$ (fractions of budget) to minimize risk while guaranteeing a minimum return.

  • Data: Mean returns $\mu$, Covariance $\Sigma \in \mathbb{S}_{++}^n$.
  • Objective: Minimize variance $x^\top \Sigma x$.
  • Constraint: Expected return $\mu^\top x \ge r_{\min}$.
  • Constraint: Full investment $\mathbf{1}^\top x = 1$ and long-only $x \ge 0$.

$ \begin{aligned} \text{minimize} \quad & x^\top \Sigma x \\ \text{subject to} \quad & \mu^\top x \ge r_{\min} \\ & \mathbf{1}^\top x = 1 \\ & x \ge 0 \end{aligned} $

This is a QP with $P = 2\Sigma \succ 0$, capturing the fundamental risk-return tradeoff in finance.

3.4 Relationship to LP

Every LP is a QP (set $P = 0$). Thus: LP ⊂ QP.

4. Problem Reformulation and Equivalence

4.1 Equivalent Problems

Two optimization problems are equivalent if the solution of one can be readily obtained from the solution of the other, and vice versa.

Common Equivalence Transformations:

  • Change of variables: Substitute $x = \phi(y)$ where $\phi$ is one-to-one
  • Slack variables: Introduce $s$ to turn $f(x) \le t$ into $f(x) + s = t$, $s \ge 0$
  • Epigraph form: Minimize $t$ subject to $f(x) \le t$ (instead of minimizing $f(x)$)
  • Eliminating equality constraints: Use $Ax = b$ to eliminate variables

4.2 Epigraph Reformulation

The problem $\min f_0(x)$ s.t. $x \in \mathcal{C}$ is equivalent to:

$ \begin{aligned} \text{minimize} \quad & t \\ \text{subject to} \quad & f_0(x) \le t \\ & x \in \mathcal{C} \end{aligned} $

This transformation is crucial for reformulating problems with complex objectives into standard forms.

4.3 Example: Minimizing $\ell_\infty$ Norm

Minimize $\|Ax - b\|_\infty$ can be reformulated as an LP:

$ \begin{aligned} \text{minimize} \quad & t \\ \text{subject to} \quad & -t \mathbf{1} \preceq Ax - b \preceq t \mathbf{1} \end{aligned} $

4.4 Eliminating Linear Equality Constraints

Given $\min f_0(x)$ s.t. $Ax = b$ and other constraints, if $\text{range}(A) = \mathbb{R}^p$, we can parameterize the solution space as:

$ x = A^\dagger b + (I - A^\dagger A) z $

where $A^\dagger$ is the pseudoinverse and $z \in \mathbb{R}^n$ is a new variable. This reduces the problem dimension.

5. Review & Cheat Sheet

Problem Classes Reference

Class Objective Constraints Key Property
LP Linear $c^\top x$ Linear inequalities ($Ax \le b$) Solution at vertex; simplex method.
QP Convex Quadratic $\frac{1}{2}x^\top P x + q^\top x$ Linear inequalities Unique solution if $P \succ 0$.
QCQP Convex Quadratic Quadratic inequalities ($x^\top P_i x + q_i^\top x + r_i \le 0$) More general than QP; often solved as SOCP.

6. Exercises

P7.1 — Diet Problem Formulation

Formulate the following problem as a standard Linear Program (LP).

You want to buy oranges and apples. Oranges cost $2 each and provide 3 units of Vit C and 1 unit of Fiber. Apples cost $1 each and provide 1 unit of Vit C and 2 units of Fiber. You need at least 10 units of Vit C and 5 units of Fiber. Minimize the total cost.

Recap & Key Concepts

  • LP Structure: Identify variables (quantities), objective (cost), and constraints (requirements).
  • Non-negativity: Physical quantities cannot be negative. This is a standard LP constraint.

Solution

Let $x_1$ be oranges and $x_2$ be apples. Objective: Minimize $2x_1 + 1x_2$. Constraints:
1. Vit C: $3x_1 + 1x_2 \ge 10$
2. Fiber: $1x_1 + 2x_2 \ge 5$
3. Non-negativity: $x_1, x_2 \ge 0$

P7.2 — Chebyshev Approximation

Given the points $(1, 2), (2, 3), (3, 5)$, find the line $y = ax + b$ that minimizes the maximum vertical distance (Chebyshev distance) to the points. Formulate this as an LP.

Solution

We want to find $a, b$ to minimize $\max_i | (a x_i + b) - y_i |$. Let the maximum error be $t$. Minimize $t$ subject to:
$-t \le (a(1) + b) - 2 \le t$
$-t \le (a(2) + b) - 3 \le t$
$-t \le (a(3) + b) - 5 \le t$
This is an LP in variables $a, b, t$.

P7.3 — Least Squares with Box Constraints

Given $A \in \mathbb{R}^{m \times n}$ and $b \in \mathbb{R}^m$, formulate the problem $\min \|Ax - b\|_2^2$ subject to $-1 \le x_i \le 1$ as a standard QP.

Solution

Objective: $x^\top A^\top A x - 2 b^\top A x + b^\top b$. Dropping constant term, minimize $\frac{1}{2} x^\top (2A^\top A) x + (-2A^\top b)^\top x$.
Constraints: $x \preceq \mathbf{1}$ and $-x \preceq \mathbf{1}$ (equivalent to $x \succeq -\mathbf{1}$).
This matches the QP form with $P = 2A^\top A, q = -2A^\top b, G = [I; -I], h = [\mathbf{1}; \mathbf{1}]$.

P7.4 — Reformulation

Show that minimizing $\|x\|_1$ subject to $Ax=b$ is equivalent to an LP.

Solution

We introduce $u \in \mathbb{R}^n$. Replace $|x_i|$ with $u_i$ and add constraints $-u_i \le x_i \le u_i$.
Minimize $\sum u_i$ subject to:
$Ax = b$
$x - u \le 0$
$-x - u \le 0$
This is an LP in variables $(x, u)$.
