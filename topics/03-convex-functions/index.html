  • How operations like addition, scaling, and composition preserve convexity.

Key Concepts

Concept 1: Convex Functions

A function $f$ is convex if its domain is a convex set and for all $x, y$ in its domain, and for any $\theta$ with $0 \le \theta \le 1$, we have $f(\theta x + (1-\theta)y) \le \theta f(x) + (1-\theta) f(y)$.

Visual intuition: The line segment between any two points on the graph of a convex function lies above or on the graph.
