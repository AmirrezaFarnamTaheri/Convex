1. Affine and Convex Sets: Definitions and Basic Properties

The geometry of optimization is built on understanding how points combine. We begin with the fundamental distinction between lines (affine) and segments (convex).

1.1 Affine Combinations and Affine Sets

An affine combination of points , \ldots, x_k$ is a weighted sum where the weights sum to one:

1316 x = \sum_{i=1}^k \theta_i x_i \quad \text{where} \quad \sum_{i=1}^k \theta_i = 1 1316

Weights can be negativeâ€”there\'s no nonnegativity requirement. Affine combinations describe lines, planes, and hyperplanes.

Definition (Affine Set)

A set $ is affine if it contains all affine combinations of its points. Equivalently, the line through any two points in $ lies entirely in $.

Affine Sets as Translated Subspaces

Every affine set is merely a linear subspace that has been shifted (translated) away from the origin.

1.2 Convex Combinations and Convex Sets

A convex combination is an affine combination with the additional constraint that all weights are nonnegative:

1316 \sum_{i=1}^k \theta_i x_i \quad \text{where} \quad \theta_i \ge 0, \ \sum_{i=1}^k \theta_i = 1 1316

This describes the "filled-in" shape between points. For two points, it describes the line segment connecting them.

Definition (Convex Set)

A set \subseteq \mathbb{R}^n$ is convex if for any two points , y \in C$ and any $\theta \in [0,1]$:

1316 \theta x + (1-\theta)y \in C 1316

Geometric meaning: The line segment connecting any two points in the set lies entirely within the setâ€”no "dents" or "holes."

1316 \mathrm{conv}(S) = \left\{\sum_{i=1}^k \theta_i x_i \ \bigg| \ x_i \in S, \ \theta_i \ge 0, \ \sum_{i=1}^k \theta_i = 1, \ k \in \mathbb{N}\right\} 1316

It is the smallest convex set containing $.

If \subseteq \mathbb{R}^n$, then every point in $\mathrm{conv}(S)$ can be written as a convex combination of at most +1$ points from $.

For example, in $\mathbb{R}^2$, any point in the convex hull of a cloud of points lies within a triangle formed by just 3 of those points.

2. Canonical Convex Sets: Building Blocks

Convex optimization relies on a library of standard convex sets. Recognizing these forms is the first step in formulating problems.

2.1 Hyperplanes and Halfspaces

A hyperplane is an affine set of dimension -1$, defined by a linear equation:

1316 H = \{x \in \mathbb{R}^n \mid a^\top x = b\} \quad (a \neq 0) 1316

A halfspace is the region on one side of a hyperplane:

1316 H^+ = \{x \in \mathbb{R}^n \mid a^\top x \le b\} 1316

Convexity: Both are convex. For a halfspace, if ^\top x \le b$ and ^\top y \le b$, then ^\top (\theta x + (1-\theta)y) = \theta (a^\top x) + (1-\theta)(a^\top y) \le \theta b + (1-\theta)b = b$.

2.2 Norm Balls and Ellipsoids

A norm ball for any norm $\|\cdot\|$ is convex:

1316 B = \{x \mid \|x - x_c\| \le r\} 1316

This follows directly from the triangle inequality (which is effectively the convexity condition for the norm function).

An ellipsoid is the image of a Euclidean ball under an affine map. Explicitly:

1316 \mathcal{E} = \{x \mid (x - x_c)^\top P^{-1} (x - x_c) \le 1\} 1316

where \in \mathbb{S}^n_{++}$ (symmetric positive definite). The eigenvalues of $ determine the squared lengths of the semi-axes.

2.3 Polyhedra

A polyhedron is the solution set of finitely many linear inequalities and equalities:

1316 \mathcal{P} = \{x \in \mathbb{R}^n \mid Ax \le b, \ Cx = d\} 1316

It is the intersection of finitely many halfspaces and hyperplanes. Since intersection preserves convexity (see Section 3), polyhedra are convex.

2.4 Cones and Important Cone Examples

A set $ is a cone if for every \in K$ and $\theta \ge 0$, we have $\theta x \in K$ (closed under nonnegative scaling).

A convex cone is a cone that is also convex. Equivalently, $ is a convex cone if for all , y \in K$ and $\theta_1, \theta_2 \ge 0$:

1316 \theta_1 x + \theta_2 y \in K 1316

The Second-Order Cone (SOC)

Also called the Lorentz cone or "Ice Cream" cone:

1316 \mathcal{Q}^{n+1} = \{(x, t) \in \mathbb{R}^{n+1} \mid \|x\|_2 \le t\} 1316

This cone is central to Second-Order Cone Programming (SOCP).


The Positive Semidefinite (PSD) Cone

The set of all positive semidefinite matrices forms a cone:

1316 \mathbb{S}^n_+ = \{X \in \mathbb{S}^n \mid X \succeq 0\} 1316

where $\mathbb{S}^n$ denotes symmetric \times n$ matrices. This is the foundation of Semidefinite Programming (SDP).

Proof: The PSD Cone is Convex

We need to show that if \succeq 0$ and \succeq 0$, then $\theta A + (1-\theta)B \succeq 0$ for $\theta \in [0,1]$.

Step 1: Definition Check. \succeq 0$ means ^\top A v \ge 0$ for all vectors $. Similarly ^\top B v \ge 0$.
Step 2: Linear Combination. Consider the quadratic form of the combination: 1316 v^\top (\theta A + (1-\theta)B) v = v^\top (\theta A) v + v^\top ((1-\theta) B) v 1316 1316 = \theta (v^\top A v) + (1-\theta) (v^\top B v) 1316
Step 3: Sign Analysis. Since $\theta \in [0,1]$, both coefficients are non-negative. Since ^\top A v \ge 0$ and ^\top B v \ge 0$, the weighted sum is non-negative.
Conclusion: ^\top (\theta A + (1-\theta)B) v \ge 0$ for all $, so the combination is PSD. Thus $\mathbb{S}^n_+$ is a convex cone.

3. Operations that Preserve Convexity

We can prove that a complex set is convex by building it from simpler convex sets using operations that preserve convexity. This is the "calculus" of convex sets.

3.1 Intersection

The intersection of any collection (finite or infinite) of convex sets is convex.

Proof

Step 3: Therefore, $\theta x + (1-\theta)y \in \bigcap_{i \in I} C_i = C$, proving $ is convex.

4. Separating and Supporting Hyperplane Theorems

The geometric heart of convex analysis lies in separation: if two convex sets don\'t touch, we can slide a sheet of paper (hyperplane) between them.

4.1 Separating Hyperplane Theorem

4.2 The Supporting Hyperplane Theorem

Theorem (Supporting Hyperplane)

Let \subseteq \mathbb{R}^n$ be a nonempty, closed, convex set, and let \in \partial C$ (the boundary of $). Then there exists a supporting hyperplane to $ at $: there exists \in \mathbb{R}^n \setminus \{0\}$ such that:

1316 a^\top x \le a^\top x_0 \quad \forall x \in C 1316

Step 1: Take an exterior point. Since \in \partial C$ (boundary), there exists a sequence $\{y_k\} \subset \mathbb{R}^n \setminus C$ with \to x_0$.
Step 2: Project each $ onto $. Let = \Pi_C(y_k)$ be the projection. By the projection characterization: 1316 (y_k - p_k)^\top (x - p_k) \le 0 \quad \forall x \in C 1316
Step 3: Pass to the limit. As \to \infty$, we have \to x_0$ (since \to x_0$ and projections are continuous). Normalize = (y_k - p_k) / \|y_k - p_k\|_2$ and extract a convergent subsequence \to a$ with $\|a\|_2 = 1$. Taking limits: 1316 a^\top (x - x_0) \le 0 \quad \forall x \in C 1316 Rearranging: ^\top x \le a^\top x_0$ for all \in C$.

5. Cones, Proper Cones, and Dual Cones

5.1 Cones and Convex Cones

A set \subseteq \mathbb{R}^n$ is a cone if $\theta x \in K$ for all \in K$ and $\theta \ge 0$ (closed under nonnegative scaling).

A convex cone satisfies both cone and convexity properties. Equivalently, $ is a convex cone if:

1316 x, y \in K, \ \theta_1, \theta_2 \ge 0 \quad \implies \quad \theta_1 x + \theta_2 y \in K 1316

5.2 Proper Cones

A cone $ is proper if it satisfies four key properties:

  1. Convex: $ is a convex cone
  2. Closed: $ contains all its limit points
  3. Pointed: \cap (-K) = \{0\}$ (no line through the origin lies in $)
  4. Solid: $\mathrm{int}(K) \neq \emptyset$ (has nonempty interior)

Proper cones can define generalized inequalities (partial orders) on $\mathbb{R}^n$.

Examples of proper cones:

  • $\mathbb{R}^n_+$ (Nonnegative orthant)
  • $\mathcal{Q}^{n+1}$ (Second-order cone)
  • $\mathbb{S}^n_+$ (PSD cone)
  • Cone of nonnegative polynomials on 1$

5.3 The Dual Cone

The dual cone of $ is defined as:

1316 K^* = \{y \in \mathbb{R}^n \mid y^\top x \ge 0 \ \forall x \in K\} 1316

It consists of all vectors that make a nonnegative inner product with every vector in $.

Properties of Dual Cones

  • ^*$ is always closed and convex (even if $ is not)
  • Bipolar theorem: If $ is a closed convex cone, then ^* = K$
  • Reverses inclusion: \subseteq K_2 \implies K_2^* \subseteq K_1^*$
  • Intersection/Sum duality: ^* = K_1^* \cap K_2^*$ and ^* = \mathrm{cl}(K_1^* + K_2^*)$

5.4 Self-Dual Cones

Some of the most important cones in optimization are self-dual: ^* = K$.

Remarkably, the Orthant, SOC, and PSD cones are all self-dual.

Nonnegative Orthant

$\mathbb{R}^n_+ = \{x \in \mathbb{R}^n \mid x \ge 0\}$ is self-dual.

Proof of Self-Duality

($\subseteq$): If \in \mathbb{R}^n_+$, then for any \in \mathbb{R}^n_+$: 1316 y^\top x = \sum_{i=1}^n y_i x_i \ge 0 1316 (since all terms are nonnegative). Thus \in (\mathbb{R}^n_+)^*$.
($\supseteq$): If \in (\mathbb{R}^n_+)^*$, then ^\top x \ge 0$ for all \ge 0$. Taking = e_i$ (standard basis vector), we get \ge 0$ for all $. Thus \in \mathbb{R}^n_+$.

Second-Order Cone

$\mathcal{Q}^{n+1} = \{(x, t) \in \mathbb{R}^{n+1} \mid \|x\|_2 \le t\}$ is self-dual.

Proof of Self-Duality

($\subseteq$): If \in \mathcal{Q}$ (so $\|y\|_2 \le s$) and \in \mathcal{Q}$ (so $\|x\|_2 \le t$), then by Cauchy-Schwarz: 1316 y^\top x + st \ge -\|y\|_2 \|x\|_2 + st \ge -st + st = 0 1316 Thus \in \mathcal{Q}^*$.
($\supseteq$): If \notin \mathcal{Q}$, then $\|y\|_2 > s$. We show \notin \mathcal{Q}^*$ by finding \in \mathcal{Q}$ with ^\top x + st < 0$:
  • If < 0$: Take = (0, 1) \in \mathcal{Q}$. Then ^\top x + st = s < 0$.
  • If \ge 0$ but $\|y\|_2 > s$: Take = (-y/\|y\|_2, 1) \in \mathcal{Q}$. Then: 1316 y^\top x + st = -\|y\|_2 + s < 0 1316

Positive Semidefinite Cone

$\mathbb{S}^n_+ = \{X \in \mathbb{S}^n \mid X \succeq 0\}$ is self-dual (with the trace inner product $\langle X, Y \rangle = \mathrm{tr}(X^\top Y)$).

However, norm cones are dual to the dual-norm cone ($ is dual to \infty$).

5.5 Generalized Inequalities

A proper cone $ defines a partial order $\preceq_K$ on $\mathbb{R}^n$:

1316 x \preceq_K y \quad \iff \quad y - x \in K 1316

This generalizes the standard componentwise order ($\mathbb{R}^n_+$ gives \le y$) and the Loewner order ($\mathbb{S}^n_+$ gives \preceq Y$).

Minimum vs. Minimal

Because the order is partial, we distinguish between:

  • Minimum: \in S$ is the minimum if \preceq_K y$ for all \in S$. (Unique if it exists).
  • Minimal: \in S$ is minimal if no \in S$ satisfies \preceq_K x$ (except $ itself).

Example: Matrix Inequalities

Consider the inequality \preceq B$ where , B \in \mathbb{S}^n$. This is defined by the PSD cone = \mathbb{S}^n_+$. 1316 A \preceq B \iff B - A \in \mathbb{S}^n_+ \iff B - A \succeq 0 1316 Concrete Example: Let = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}$ and = \begin{bmatrix} 2 & 1 \\ 1 & 2 \end{bmatrix}$. 1316 B - A = \begin{bmatrix} 1 & 1 \\ 1 & 1 \end{bmatrix} 1316 Is - A \succeq 0127 The eigenvalues of $\begin{bmatrix} 1 & 1 \\ 1 & 1 \end{bmatrix}$ are $ and -bash$. Since they are non-negative, - A$ is PSD. Thus \preceq B$.
Note that generalized inequalities are partial orders. It is possible that neither \preceq Y$ nor \preceq X$ holds (matrices can be incomparable).

Properties of Generalized Inequalities

  • Reflexive: \preceq_K x$ (since -bash \in K$)
  • Antisymmetric: \preceq_K y$ and \preceq_K x \implies x = y$ (by pointedness)
  • Transitive: \preceq_K y \preceq_K z \implies x \preceq_K z$ (by convexity)
  • Preserved by addition: \preceq_K y \implies x + z \preceq_K y + z$
  • Preserved by nonnegative scaling: \preceq_K y, \alpha \ge 0 \implies \alpha x \preceq_K \alpha y$

6. Topological Toolkit: Closure, Interior, Relative Interior

To work precisely with convex sets and optimality conditions, we need clear notions of "inside," "edge," and "outside."

6.1 Basic Topological Definitions

Let \subseteq \mathbb{R}^n$ be any set.

  • Closure $\mathrm{cl}(C)$: The set of all limits of convergent sequences in $. It\'s the smallest closed set containing $.
  • Interior $\mathrm{int}(C)$: Points with some open ball fully contained in $. Intuitively, "strictly inside" $.
  • Boundary $\partial C = \mathrm{cl}(C) \setminus \mathrm{int}(C)$: The "edge" of $.
  • Affine hull $\mathrm{aff}(C)$: The smallest affine set containing $.
  • Relative interior $\mathrm{ri}(C)$: The interior of $ relative to $\mathrm{aff}(C)$. Points strictly inside when viewed in the affine hull.

Example 1: Line Segment in $\mathbb{R}^2$

Consider = \{(t, 0) \mid 0 \le t \le 1\}$ (a line segment on the himBHsaxis).

  • $\mathrm{int}(C) = \emptyset$ (no open ball in $\mathbb{R}^2$ fits inside a line)
  • $\mathrm{aff}(C) = \{(t, 0) \mid t \in \mathbb{R}\}$ (the entire himBHsaxis)
  • $\mathrm{ri}(C) = \{(t, 0) \mid 0 < t < 1\}$ (interior relative to the himBHsaxis)

Example 1b: A Flat Disk in 3D

Consider a flat circular disk floating in 3D space: = \{(x, y, 0) \mid x^2 + y^2 \le 1\}$.

  • Interior: Empty ($\emptyset$). A 3D ball has volume, but this disk is flat, so no 3D ball fits inside it.
  • Affine Hull: The himBHsplane (=0$).
  • Relative Interior: The disk without its boundary circle: $\{(x, y, 0) \mid x^2 + y^2 < 1\}$. Viewed "relative" to the plane it lives in, it has an interior!

Example 2: Standard Simplex

$\Delta^n = \{x \in \mathbb{R}^n \mid x \ge 0, \mathbf{1}^\top x = 1\}$.

  • $\mathrm{int}(\Delta^n) = \emptyset$ (lies in a hyperplane)
  • $\mathrm{aff}(\Delta^n) = \{x \mid \mathbf{1}^\top x = 1\}$ (the hyperplane)
  • $\mathrm{ri}(\Delta^n) = \{x > 0 \mid \mathbf{1}^\top x = 1\}$ (all coordinates strictly positive)

6.2 Key Properties for Convex Sets

Facts About Convex Sets

  1. $\mathrm{cl}(C)$, $\mathrm{int}(C)$ (if nonempty), and $\mathrm{ri}(C)$ are all convex when $ is convex.
  2. If $ is convex with nonempty interior, then $\mathrm{cl}(C) = \mathrm{cl}(\mathrm{int}(C))$ and $\mathrm{int}(C) = \mathrm{int}(\mathrm{cl}(C))$.
  3. Relative interior of intersection: If $\mathrm{ri}(C) \cap \mathrm{ri}(D) \neq \emptyset$, then: 1316 \mathrm{ri}(C \cap D) = \mathrm{ri}(C) \cap \mathrm{ri}(D) 1316
  4. Affine functions: If (x) = Ax + b$, then (\mathrm{ri}(C)) = \mathrm{ri}(f(C))$ for convex $.

6.3 Why Relative Interior Matters

Relative interior is crucial for:

  • Constraint qualifications: Slater\'s condition for strong duality (Lecture 05) requires a point in $\mathrm{ri}(\mathrm{dom}(f_0))$
  • Optimality conditions: KKT conditions require regularity at boundary points
  • Interior-point methods: Algorithms that approach optimality from the relative interior

6.4 Indicator Functions and Sublevel Sets

A convex set $ can be represented by its indicator function $\delta_C(x)$ (0 if \in C$, $\infty$ otherwise). $\delta_C$ is a convex function.

8. Quick Reference Glossary

  • Affine set: Contains lines through any two points.
  • Convex set: Contains segments between any two points.
  • Convex hull: Smallest convex set containing $.
  • Hyperplane: $\{x \mid a^\top x = b\}$.
  • Halfspace: $\{x \mid a^\top x \le b\}$.
  • Polyhedron: Intersection of finite halfspaces.
  • Cone: Closed under positive scaling.
  • Proper cone: Convex, closed, pointed, solid.
  • Dual cone: $\{y \mid y^\top x \ge 0 \forall x \in K\}$.
  • Separating Hyperplane: Plane between disjoint convex sets.
