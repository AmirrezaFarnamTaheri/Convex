Key Concepts

Concept 1: Convex Sets

A set $\mathcal{C}$ is convex if for any $x_1, x_2 \in \mathcal{C}$ and any $\theta$ with $0 \le \theta \le 1$, we have $\theta x_1 + (1-\theta) x_2 \in \mathcal{C}$.

Visual intuition: The line segment between any two points in a convex set is contained within the set.
